# 📡 Echo Notify

> **极简、双核、高效的跨设备推送神器**
>
> *Designed for Perfectionists. @author Yuehuaer*

**Echo Notify** (原 Bark Crystal) 是一款专为极客打造的 Chrome 扩展。它摒弃了繁杂的设计，采用 **800x600 黄金比例**的“Max Vision”视界，融合深色毛玻璃（Glassmorphism）美学。

你的每一次输入，都会在另一端产生精准的“回声”。

# ✨ 核心特性 (Core Features)

### 1. 🔄 双核推送引擎 (Dual Engine)
- **Bark 模式**：支持原生 Bark 协议，具备 Markdown 渲染、分组、角标等全功能。
- **助手模式 (Helper)**：完美适配“消息推送助手”，支持标题、副标题、图片、角标等参数，兼容性更强。
- **一键切换**：点击顶部 Logo 旁的按钮，即可在两种服务间秒切，配置自动隔离，互不干扰。

### 2. 📱 沉浸式手机预览 (Live Preview)
- **所见即所得**：右侧内置 iOS 风格手机预览窗，实时渲染标题、正文和图片。
- **底部 Tab 导航**：采用原生 iOS 底部导航栏设计，在 **[👁️ 预览]** 和 **[📜 历史]** 之间优雅切换。

### 3. 💾 暴力草稿箱 (Instant Save)
- **毫秒级自动保存**：无论你输入了一个字还是粘贴了一张图，内容都会瞬间存入本地。
- **防丢失机制**：误触关闭、浏览器崩溃？没关系，再次打开插件，一切都在原处。

### 4. 📝 智能 Markdown (Bark 模式限定)
- **Smart MD**：开启开关后，插件会自动优化换行逻辑（自动将回车转换为双换行），让手机端排版更整洁。
- **语法高亮**：预览窗内置 Markdown 解析引擎，实时预览粗体、代码块、引用等效果。

### 5. 🎨 全参数支持
- **支持高级参数**：`分组 (Group)`、`角标 (Badge)`、`图标 (Icon)`、`图片 (Image)`、`副标题 (Subtitle)`
- **一键抓取**：支持一键抓取当前网页的 **标题**、**链接** 和 **Favicon图标**，并自动填入对应字段。
- **一键清空**：标题栏右侧设有 **[🧹]** 按钮，一键重置所有输入，开始新的推送。
- **历史回填**：在历史记录 Tab 中点击任意条目，内容瞬间回填至编辑区，方便二次编辑。

## 🚀 安装指南 (Installation)

由于本项目尚未上架 Chrome 商店，请使用“开发者模式”安装：

1. **下载代码**：点击右上角 `Code` -> `Download ZIP` 并解压，或使用 Git 克隆本仓库。
2. **打开扩展管理**：在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
3. **开启开发者模式**：打开右上角的 **“开发者模式”** 开关。
4. **加载插件**：点击左上角的 **“加载已解压的扩展程序”**。
5. **选择文件夹**：选择包含 `manifest.json` 的 **Bark Crystal** 文件夹。
6. **完成**：现在你应该能在浏览器右上角看到 💎 图标了！

## ⚙️ 配置说明 (Configuration)

点击插件右上角的 **设置 (⚙️)** 按钮进行配置。插件会自动根据当前模式显示对应的配置项：

| 模式 | 需填写的参数 | 获取方式 |
| :--- | :--- | :--- |
| **Bark** | `Device Key`, `Server URL` | Bark APP获取 |
| **消息推送助手** | `User ID`, `User Key` | 消息推送助手APP获取 |

*注：配置数据仅保存在您本地的 Chrome 存储中，绝不上传至任何第三方服务器。*

## 📖 Markdown 使用技巧

| 功能 | 语法示例 | 智能模式表现 |
| :--- | :--- | :--- |
| **一级标题** | `# 标题内容` | 自动加粗、加大、下划线 |
| **列表** | `- 列表项` | 自动处理缩进和圆点 |
| **引用** | `> 引用文本` | iOS 风格灰色竖线背景 |
| **链接** | `[文字](URL)` | 蓝色高亮 |
| **图片** | `![](图片URL)` | 在预览区显示缩略图 |
| **换行** | `回车键` | **自动转换为双换行 (推荐开启)** |
